Class: Collection

Classes_API This module contains classes wrapping objects returned by the PSS Fleet Data API.~Collection(collectionMetadata, alliances, users)

Represents a PSS Fleet Data Collection. This is a snapshot of top 100 fleets and their members and top 100 players not in a fleet taken once an hour.
